# Ubuntu Terminal Simulator

A web-based Ubuntu terminal simulator for learning Linux commands.

## Features
- Interactive terminal
- Learn basic Linux commands
- Safe environment - no real commands executed
- Responsive design

## Commands Available
- `help` - Show all commands
- `ls` - List files
- `pwd` - Show current directory
- `date` - Show current date/time
- `whoami` - Show current user
- `clear` - Clear terminal

## How to Use
1. Type commands in the input field
2. Press Enter to execute
3. Click on commands in the reference panel to insert them
